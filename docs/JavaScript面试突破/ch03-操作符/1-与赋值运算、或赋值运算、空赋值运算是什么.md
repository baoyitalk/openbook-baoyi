
# 逻辑 或赋值运算符的含义


## 什么是假值
### 8个特殊值
```js
false       // 布尔假
0           // 数字零
-0          // 负零
0n          // BigInt 零
''          // 空字符串
null        // 空对象引用
undefined   // 未定义
NaN         // 非数字

```

### 误区
```
// 这些看起来"空"但其实是真值
Boolean([])        // true  ← 空数组
Boolean({})        // true  ← 空对象
Boolean('0')       // true  ← 字符串'0'
Boolean('false')   // true  ← 字符串'false'

```



**一句话：假值就是 JS 认为"等同于没有"的 8 个特殊值。**

## 1-  或赋值运算x ||= y等价代码是什么

如果x为假 ，将y赋值给x，即：


```js
if(!x) {
   x = y
}
```


案例1  逻辑 或赋值 ||=的应用

```js
// 逻辑或赋值 ||=的应用
const a = { count: 50, title: ''};
  
a.count ||= 10

console.log(a.count); // 50

a.title ||= 'title is empty'

console.log(a.title); // 'title is empty'

```




## 2- 与赋值运算x &&= y 等价代码是什么



逻辑与赋值运算符&&= 的含义是： 如果x为真，将y赋值给x，即

```js
if(x) {
  x = y
}
```

案例 逻辑与赋值运算符&&= 的应用

```js
// 逻辑与赋值运算符 &&= 的应用

  
  

const a = { count: 50, title: ''};

  

a.count &&= 10

  

console.log(a.count); // 10

  

a.title &&= 'title is empty' // a.title是空属于 假值 不赋值

console.log(a.title); // 打印‘’
```


## 3- 空赋值运算??=等价代码是什么

逻辑空赋值运算符 x ??= y 的含义是：如果x为空值 （null 或 undefined），将y赋值给x

```js
if(x === null || x === undefined) {
   x = y
}
```


逻辑空赋值运算符??=的应用

```js
// 逻辑空赋值运算符 ??=的应用

  

const a = { count: 50, title: ''};

  

a.count ??= 10

  

console.log(a.count); // 50

  

a.title ??= 'title is empty' // a.title是空属于 假值 但不符合 特殊假值 null、undefined 不赋值

console.log(a.title); // 打印‘’

```

误区：

- `??=` 只在左边的值是 `null` 或 `undefined` 时，才会把右边的值赋给左边；
- 哪怕左边是 `''`（空字符串）、`0`、`false` 这些 “假值”，只要不是 `null/undefined`，`??=` 都**不会执行赋值**。
