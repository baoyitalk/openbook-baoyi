> **技术栈**: React + Docusaurus

> **创建时间**: 2025-12-16

## 📚 支柱体系概述

本项目采用精简的支柱体系，适配电子书项目特点：

### 支柱结构

```
__pillars__/
├── README.md                    ← 本文件
├── AI_QUICK_START.md            ← AI 快速启动指南
│
├── 01-product/                  ← 产品支柱（内容规划）
│   ├── README.md
│   ├── content-plan/            ← 内容规划
│   └── baseline/                ← 生产基线
│
├── 02-architecture/             ← 架构支柱（技术配置）
│   ├── README.md
│   ├── docs/                    ← 技术文档
│   └── baseline/                ← 生产基线
│
├── 04-operations/               ← 运维支柱（部署运维）
│   ├── README.md
│   ├── docs/                    ← 运维文档
│   └── baseline/                ← 生产基线
│
└── 05-iteration/                ← 迭代支柱
    ├── README.md
    ├── CURRENT_VERSION.md       ← 当前版本状态
    └── v1.0.0/                  ← 版本目录
```

---

## 🎯 各支柱职责

### 01-product - 产品支柱
- 电子书内容规划
- 章节结构设计
- 目标读者定位
- 内容更新计划

### 02-architecture - 架构支柱
- Docusaurus 配置
- 主题定制方案
- 插件使用说明
- 技术决策记录

### 04-operations - 运维支柱
- 部署流程文档
- 发布检查清单
- 域名和托管配置

### 05-iteration - 迭代支柱
- 版本迭代记录
- 开发日志
- 内容更新记录
