## 🔄 回滚方案

如果发现问题，可以快速回滚：

### GitHub Pages
```bash
# 回滚到上一个版本
git revert HEAD
git push origin gh-pages
```

### Vercel/Netlify
- 在控制台选择上一个部署版本
- 点击回滚按钮
