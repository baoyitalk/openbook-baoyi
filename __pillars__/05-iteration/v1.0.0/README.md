## 📝 重要决策

### 技术选型
- **框架**: Docusaurus v3.9.2
- **UI 库**: React 18.3.1
- **包管理器**: npm
- **开发服务器**: http://localhost:3000
- **部署平台**: 待定（GitHub Pages / Vercel / Netlify）

### 架构决策
- 采用 Docusaurus Classic 预设，包含文档和博客功能
- 使用中文作为默认语言
- 支持深色模式切换
- 采用响应式设计，适配移动端

### 已实现功能
- ✅ 文档系统 - 支持多层级文档结构
- ✅ 博客系统 - 支持标签、作者、分页
- ✅ 首页设计 - 展示项目特点和快速开始
- ✅ 导航栏 - 文档、博客、GitHub 链接
- ✅ 侧边栏 - 自动生成文档导航
- ✅ 代码高亮 - 支持多种编程语言
- ✅ 主题切换 - 浅色/深色模式
