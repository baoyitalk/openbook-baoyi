## 备注

### 支柱体系说明
本项目采用精简的支柱体系，适配 Docusaurus 电子书项目特点：
- **01-product**: 内容规划和章节结构
- **02-architecture**: 技术配置和架构决策
- **04-operations**: 部署运维和发布管理
- **05-iteration**: 版本迭代和开发日志

不需要 03-testing 测试支柱，因为电子书项目主要是内容管理，不需要复杂的测试体系。

### 技术栈
- **核心框架**: Docusaurus v3.9.2
- **UI 库**: React 18.3.1
- **包管理器**: npm
- **开发服务器**: http://localhost:3000

### 项目结构
```
openbook-baoyi/
├── docs/                    # 文档内容
├── blog/                    # 博客文章
├── src/                     # 自定义组件
│   ├── pages/              # 自定义页面
│   └── css/                # 样式文件
├── static/                  # 静态资源
├── docusaurus.config.js     # 配置文件
├── sidebars.js             # 侧边栏配置
└── package.json            # 依赖管理
```

### 功能特点
- ✅ 文档系统 - 支持多层级文档结构
- ✅ 博客系统 - 支持标签、作者、分页
- ✅ 响应式设计 - 完美适配移动端
- ✅ 深色模式 - 支持主题切换
- ✅ 代码高亮 - 支持多种编程语言
- ✅ 中文支持 - 默认中文界面
