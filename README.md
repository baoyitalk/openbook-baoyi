# 开源电子书项目

> 基于 Docusaurus 的开源电子书项目  
> **当前版本**: v1.0.0（开发中）

---

## 📚 项目简介

本项目是一个基于 Docusaurus 的开源电子书项目，采用 React 技术栈，提供简洁、美观、易用的阅读体验。

---

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm start
```

### 构建生产版本
```bash
npm run build
```

### 预览生产构建
```bash
npm run serve
```

## 📋 开发规范

### Git 分支管理
- `main` - 主分支（生产环境）
- `dev-v{X.X.X}` - 开发分支

### Git 提交规范
- `feat:` - 新功能
- `fix:` - 修复
- `docs:` - 文档更新
- `chore:` - 构建/配置更新
- `refactor:` - 代码重构
- `style:` - 代码格式调整

### 开发流程
1. 基于 main 创建开发分支
2. 在开发分支进行开发
3. 完成后提交 PR
4. 代码审查通过后合并到 main
5. 打 tag 发布版本
